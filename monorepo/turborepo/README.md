# monorepo

Monorepo是一种项目管理方式，在它之前，管理方式是MultiRepo，即每个项目对应一个仓库。这会导致一些弊端，比如代码管理分散、相同功能在每个仓库重复维护等问题。

常见Monorepo工具有：

| 工具      | 简述 |
| ----------- | ----------- |
| Bit      | 用于组件驱动开发的工具链  |
| Turborepo   | 用于 JavaScript 和 TypeScript 代码库的高性能构建系统        |
| Nx   | 用于开发和维护大型前端应用程序的工具集        |
| Lerna   | 用于管理包含多个软件包的JavaScript项目的工具        |
| Rush   |         |
