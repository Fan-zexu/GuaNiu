# 在函数中定义泛型

取最大值函数合并

```rs
fn largest_i32(list: &[i32]) -> &i32 {
    let mut largest = &list[0];
    for item in list {
        if item > largest {
            largest = item;
        }
    }
    largest
}


fn largest_char(list: &[char]) -> &char {
    let mut largest = &list[0];

    for item in list {
        if item > largest {
            largest = item;
        }
    }

    largest
}
```

通过泛型，消除重复

```rs
fn largest<T>(&list: &[T]) -> &T
```
