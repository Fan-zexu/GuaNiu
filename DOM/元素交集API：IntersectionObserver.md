# IntersectionObserver Api

它是浏览器提供的一个用来对比元素是否存在交集的API，兼容性不错。

> [参考MDN](IntersectionObserver.api)

## 使用场景

- 懒加载场景，比如图片

- 无线滚动

- 模块曝光检测
